CSS

Cascading Style Sheets，层叠样式表。是用来描述网页内容的表现（外观）的语言。

CSS 的基本结构

CSS 的基本结构由选择器、属性和属性值组成：

  • 选择器：指定要应用样式的 HTML 元素，比如标签、类名、ID 等。
  • 属性：指定要设置的样式类型，如color、font-size、margin等。
  • 属性值：为属性指定具体的值，如red、16px、10px等。

CSS 常用选择器

  1. 元素选择器：直接选择 HTML 标签。

  2. 类选择器：选择拥有特定类名的元素，类选择器以.开头。

  3. ID 选择器：选择具有特定 ID 的元素，ID 选择器以#开头。

  4. 后代选择器：选择某个元素的后代元素。

  5. 通配符选择器：选择所有的元素。

2. Flexbox 布局

Flexbox 是 CSS3 新增的布局方式，通过定义容器和子元素的属性，可以轻松实现各种复杂布局。

3. Grid 布局

Grid 是另一种非常强大的布局方式，它允许我们将页面分成多个网格，元素可以放置在这些网格中。

CSS 预处理器

随着前端开发的复杂度提升，传统的 CSS 有时显得冗长和难以维护。这时，CSS 预处理器如Sass、LESS、Stylus等应运而生，它们通过引入变量、嵌套规则、混合宏等特性，使得 CSS 的编写变得更加简洁和高效。
